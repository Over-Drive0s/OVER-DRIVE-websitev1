# Overdrive IO

Multi-page corporate website for [overdriveio.com](https://overdriveio.com) — operational software consulting and engineering.

## Stack

- React 19 + TypeScript
- React Router 7
- Vite 6
- Tailwind CSS 4
- lucide-react

## Pages

| Route | Page |
|---|---|
| `/` | Home |
| `/platform` | Platform |
| `/solutions` | Solutions |
| `/resources` | Resources |
| `/company` | Company |
| `/pricing` | Pricing |
| `/signin` | Sign In |
| `/request-demo` | Request Demo |

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
